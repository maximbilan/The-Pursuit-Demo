Transitions2d v1.0 for Marmalade / Airplay SDK

Currently supports the following 2D transitions effects:

- crossfade
- slide left/right/up/down

Installation:

Copy transitions2d folder to Marmalade\5.x\modules\

Note:

TransitionsExample demonstrates how you can use this module. This module has been used in the Tarantula vs Snake game for the iPhone. You can download Tarantula vs Snake from the App Store to see how this module works in practice.
